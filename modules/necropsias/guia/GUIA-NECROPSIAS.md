# Guía paso a paso — Módulo Necropsias

Esta guía explica el flujo del módulo de Necropsias: conceptos, flujo de uso, y secciones web (Registro, Listado, Comparativo). El módulo se encuentra en la **Sección 5** del portal de sanidad.

## Conceptos del módulo

### Necropsia

Es la evaluación de aves en granja, mediante la revisión de sistemas (digestivo, respiratorio, inmunológico, etc.), documentando hallazgos, lesiones y conclusiones. Cada registro incluye granja, galpón, campaña, fecha, edad del lote y detalle por sistema.

### Sistemas

Son los **bloques de evaluación** de la necropsia. Cada pestaña del registro corresponde a un sistema (por ejemplo: Sistema Inmunológico, Sistema Digestivo, Sistema Respiratorio, Evaluación Física, Diagnóstico Presuntivo).

### Niveles

Los **niveles** son las categorías dentro de cada sistema (por ejemplo: Índice Bursal, Mucosa de la Bursa, Hígado, Pododermatitis). Cada nivel agrupa varias opciones de evaluación.

### Subniveles

Son las **subcategorías** de cada nivel. Cada subnivel se evalúa marcando casillas (columnas 1 a 5, que representan aves o muestras). A medida que se marcan las casillas de un subnivel, se incrementa el **porcentaje** asociado (cada casilla equivale a 20 %).


## Flujo de uso

| **1. Crear mi necropsia** — Defina las características de la necropsia en la sección [link:registro] (5.1 Registro).
| **2. Ver mis necropsias** — Para ver las necropsias registradas use la sección [link:listado] (5.2 Listado).
| **3. Comparativo** — Para comparar distintas necropsias registradas use la sección [link:reporte] (5.3 Comparativo).


## Registro

Para registrar una necropsia vaya a la sección [link:registro] (5.1 Registro).

1. Ingrese la **fecha** y la **granja**; el sistema cargará automáticamente la **campaña** y la **edad**.
2. Complete la información para **cada sistema** (use las pestañas: Inmunológico, Digestivo, Respiratorio, Evaluación Física).
3. En cada sistema hay **niveles** (categorías). Cada nivel tiene **subniveles** (opciones). A medida que marque las casillas de cada subnivel (columnas 1 a 5), se incrementará el porcentaje en 20 % por casilla.
4. También puede ingresar **observaciones** y adjuntar **imágenes** por nivel.
5. Al finalizar use [btn:guardar-necropsia] para guardar.

| Al finalizar podrá: ver cabecera y detalles, generar reporte en formato tabla, generar reporte de imágenes y modificar la necropsia desde el listado.

## Listado

Para ver las necropsias registradas vaya a la sección [link:listado] (5.2 Listado). Use el buscador o [btn:filtrar] para localizar registros.

Por cada necropsia puede:

| [i:eye] Ver cabecera y detalles
| [i:pdf] Generar reporte en formato tabla
| [i:images] Generar reporte de imágenes
| [i:edit] Modificar la necropsia
| [i:trash] Eliminar la necropsia [i:warning] (confirmación previa)

## Comparativo

Para generar el reporte comparativo vaya a la sección [link:reporte] (5.3 Comparativo).

Seleccione un **periodo de tiempo**, **granjas** y **galpones** para comparar la información de las necropsias registradas. Útil para análisis de tendencias y hallazgos recurrentes.

| Filtros: [i:filter] periodo, granja, galpón, campaña
| Acciones: [btn:filtrar] aplicar filtros, [btn:generar-reporte] y [btn:limpiar]
